# Kleinanzeigen Ultimate Hider

A Tampermonkey userscript that enhances your [Kleinanzeigen.de](https://www.kleinanzeigen.de) browsing experience.

## Features

- **Hide Ads** - Manually hide ads you're not interested in
- **Keyword Filtering** - Automatically hide ads containing specific keywords (e.g., "Tauschwohnung")
- **Contact Tracking** - Mark ads you've already contacted
- **Dark Mode** - Toggle between light and dark themes
- **Clean Interface** - Removes empty ad placeholder slots

Hidden ads are replaced with compact placeholders that can be restored with one click.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click on `KleinanzeigenHider.user.js` and select "Raw"
3. Tampermonkey will prompt you to install the script

## Usage

- Hover over any ad to see the **hide** (eye) and **contacted** (envelope) buttons
- Click the green **Filter** button in the bottom-right corner to:
  - Add/remove filter keywords
  - View hidden ads by category
  - Toggle dark mode
  - Restore hidden ads

## Compatibility

Works with both the classic and new Tailwind-based Kleinanzeigen.de design.
